MISSION & VISION

MISSION:

To be a source of assistance for the complications faced by people in the less trodden path and to be their pillar of strength against all odds in providing them a better solution.

VISION:

Toward creating a self sustaining and balanced society which keeps getting better on its own, which is possible only when it begins from every individual.

SIRAGUGAL WAY

Anything to be done by us towards the society follows the “SIRAGUGAL WAY”. Here, we have formulated the process “Project :Dawn to Dusk”, which gives a detailed explanation on the initiation, approval, execution and documentation of the project.

List of steps of project from approach to end
  • Project Approach
  • Discussion towards approval
  • Pre Project Report
  • Defining the Project
  • Fund raising
  • Execution
  • Detailed Project Report
Project Approach
  • The Initiation of Project to be executed starts here.
  • Any person interested in bringing changes in this society can propose a project.
  • Project Approach can be done only through online.
  • The field for Project Approach will be provided in the Website.
  • The Person willing to propose a project should fill the Project Approach Form (PAF) provided in the site.
  • In PAF, the person should fill all the mandatory fields mentioned which includes Category, Beneficiary of the project, Location and basic details of the project.
  • On submission of the completed form, the project proposal will be forwarded to the Board of Management.
Review towards approval
  • The PAF submitted will be reviewed by the Board of Management for the approval of the proposed project.
  • Each Member of the Boardwill review the PAF individually and report their views towards the proposed project to Managing Trustee.
  • The Managing Trustee with the consideration of reviews from the board members has the power to proceed / hold / revise / refuse the proposed project.
  • If to be proceeded, intimation will be forwarded to the Project Proposer and District Co-Ordinator along with computer generated Project ID.
  • If to be on hold, the Project Proposer will be intimated with reasons for holding the proposed project.
  • If to be revised, the Project Proposer will be intimated to revise the idea with detailed explanation.
  • If to be refused, the Project Proposer will be intimated the declination of the proposed project with or without explanation.
  • The PAF submitted shall be reviewed within Eight days from the date of Project Proposal.
  • Intimation to proceed is not a confirmation for approval. The approval still depends on detailed Pre-Project Report (PPR).
  • The Reviews of the Board Members towards the proposed project is based on their idea, but the Board of Management needs clear picture regarding the area of approached project which will be provided by the Pre-Project Report (PPR) prepared by the Regional Co-Ordinator.
Pre Project Report
  • Pre Project Report is a report by hand which gives the detailed report of the proposed project.
  • The Proposed project will be forwarded to the District Co-Ordinator of the concerned location of the project.
  • District Co-Ordinator forwards the Proposed project to the Corresponding Regional Co-Ordinator to prepare the Pre-Project Report (PPR).
  • Regional Co-Ordinator inspects the location in person to ensure the correctness of the data provided by the project proposer and analyses the idea of approached project regarding its feasibility.
  • Regional Co-Ordinator prepares the Pre-Project Report (PPR)and forward to the District Co-Ordinator.
  • Finally, District Co-Ordinator forwards the duly signed PPR to the Board of Management.
Defining the Project
  • The approval of PPR is given by Managing Trustee and Financial Trustee of the Board.
  • Post the approval of PPR, the project will be defined by the Board of Management.
  • Definition of the Project includes Project ID, Project Name, Project Grade and Project Duration.
  • Project Flow Document which includes the ideas for the flow of project will be made ready before fund raising.
Fund Raising
  • Once the Project is defined, based on the funds available, the board will decide whether to raise funds or to proceed the project with the available funds.
  • If it is required to raise funds for that project, then the proposed project will be displayed in the website for a Fund raise campaign under Donation section.
  • The Board will fix a time limit for the completion of Fund raise campaign and authorised members from the trust will be appointed towards monitoring of the Fund raise campaign.
  • Once the funds have been raised,the Board will sanction an initial mobilization amount (certain percentage of the project estimate) to the respective Regional Co-ordinator with the Knowledge of District Co-Ordinator.
  • If the Fund raise campaign fails within the stipulated time, then the Board will take necessary steps to complete it.
Execution
  • Once the Initial Mobilization amount has been sanctioned to the Project Co-Ordinator, the execution process gets started.
  • Project Co-Ordinator takes the sole responsibility on the execution of the entire project with the available resources in the respective region of the District.
  • Project Co-Ordinator can execute the project with the idea as reported in the Project Flow Document.
  • Project Co-Ordinator should report the daily progress to the Regional Co-Ordinator.
  • The Board has the power to hold, monitor or dismiss the project execution progress at any level with proper intimation and reason.
Detailed Project Report
  • Regional Co-Ordinator prepares the Detailed Project Report of the project executed.
  • The report comprises the entire process of the project which includes the idea of execution, beneficiaries, Expense bills, Photos, Volunteers details etc…
  • The report shall be forwarded to the Board of Management with the approval of District Co-Ordinator.
  • The Board will sanction the full funds only after the submission of the Detailed Project Report.
  • Finally, the detailed project report will be displayed in the website.
